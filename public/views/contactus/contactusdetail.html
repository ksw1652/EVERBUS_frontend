전체 선택 초기화 선택 삭제
  본 어플리케이션에서는 '길찾기' 서비스 이용시 사용자의 위치를 지도 상에 표시합니다.

  이 때 사용자의 위치정보가 이용되며, 원하지 않는 경우 위치 정보 사용을 거부 할 수 있습니다.

  위치정보 사용을 거부 하더라도 어플리케이션의 대부분의 기능을 사용 하실 수 있습니다.

1. 사용자의 위치정보를 이용하는 서비스명 : 길찾기

2. 사용자의 위치정보 이용 과정 : 길찾기 서비스를 통해 사용자가 경로를 검색하게 되면 화면상에 지도가 나타나며 Google maps API v3 을 통해 사용자의 위치가 자동으로 표시됩니다.

3. 사용자의 위치 정보의 보유 및 이용기간 : 위치정보는 수집되지 않으며 서비스 제공의 목적으로만 사용됩니다.

4. 서비스 제공자 : 아이에프(I.F)
(makeyourif@gmail.com)
  에버버스는 각 지자체의 버스 정보시스템에서 실시간 노선정보와 정류소정보를 받아와 동작합니다.

  따라서 지자체의 시스템이 변경 될 경우, 실시간 데이터를 표시하는데 문제가 생길 수 있습니다.
